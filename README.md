Projeto: Consulta e Download de Faturas Claro
Descrição do Projeto

Este projeto tem como objetivo a criação de um aplicativo web para resolver o problema do envio ineficiente de faturas da Claro, focando principalmente na consulta e download de faturas em atraso.

O sistema otimiza o acesso dos clientes à informação, tornando o processo mais ágil e prático. Desenvolvido utilizando HTML e Python (Flask), o aplicativo permite que o cliente faça login com e-mail e senha. Após a autenticação, o usuário é direcionado para uma área onde é notificado sobre faturas pendentes e pode baixá-las com um clique.

A interface foi projetada para ser simples e intuitiva, com elementos visuais inspirados no site oficial da Claro, como o uso de cores predominantes e disposição clara dos campos.

 Objetivos

Facilitar a consulta de faturas em atraso

Melhorar a experiência do cliente

Simplificar o acesso às informações de contas pendentes

Servir como base para futuras melhorias, como integração com o sistema real de faturas da Claro

Tecnologias Utilizadas

HTML5 e CSS3 – front-end e interface

Python + Flask – backend e lógica do sistema

SQLite – armazenamento local de dados de login e faturas

Git & GitHub – controle de versão

 Funcionalidades

Login seguro com e-mail e senha

Área de usuário com notificações de faturas em atraso

Download de faturas com um clique

Interface intuitiva e inspirada no layout oficial da Claro

Base pronta para futuras integrações com sistemas reais
